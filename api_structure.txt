API Notify

> регистрация пользователя
> редактирование данных о пользователе
> проверка наличия пользователя 
	* получение логина и пароля
> получение списка <~> для данного пользователя 
	~ : каналы, пользователи, новости, чаты
> получение списка чатов/каналов по выбранной категории
> создание/удаление канала, поста, чата, группы

примерный формат строки запросов (TheGorazio)
	85.30.249.228/api?type=<один из типов запросо см. ниже>&<перечень параметров характерных для каждого типа запросов>?username=<очевидно>


Типы запросов:
> регистрация пользователя (signin) {
	еmail, login, password }
> редактирование пользователя (edit_profile) {
	поля пользователя с новыми значениями и имя };
> получение списка чего - либо (get) {
	тип нужной инфы, название канала/чата, категория }
> создание чего - либо (сreate) {
	тип создаваемого, название, имя пользователя, доп параметры для чата или канала }
> удаление чего - либо (delete) {
	тип, название, имя пользователя (если хозяин, то ок) }
